# DroneVision Wildlife Detection System

## Overview
DroneVision is an AI-powered wildlife monitoring system that processes drone footage using YOLOv8 object detection to identify and analyze animals in real time.

## Features
- Real-time wildlife detection
- YOLOv8 object recognition
- Professional custom bounding box annotations
- Species detection analytics summary
- Confidence-based filtering

## Tech Stack
- Python
- OpenCV
- Ultralytics YOLOv8

## Project Structure
DroneVision/
│── src/main.py
│── videos/
│── outputs/
│── requirements.txt
│── README.md

## Installation
pip install -r requirements.txt

## Run Project
python src/main.py
